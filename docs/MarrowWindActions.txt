-Study Map / Navigate
Represents studying the map, checking bearings, comparing charts, plotting the route, or otherwise correcting the ship's course
Critical Failure: Course Meter +0
Failure: Course Meter +2
Success: Course Meter +4
Critical Success: Course Meter +5
Course Meter is capped at 12
Resulting Course State is shown to players whether the check succeeds or fails
If this action resolves, skip the normal end-of-turn Course Meter decay for that turn
Navigator's Tools or Cartographer's Tools proficiency rolls with advantage
Sailor and Pirate Backgrounds do not grant advantage by default
Perception is not a core navigation substitute
-Examine the Bilge Sounding Rod for Water Levels

Total Ingress is also revealed when Water Level is known two turns in a row
Updates the total amount of each resource available to the player.

1 Labor - 1 Player - 1 Turn
-Man the Helm
Survival DC 12
Used when mast and rudder are both functional
Critical Success: Apply one Course State better for this turn only and improve the Course Meter to the next better Course State
Success: Apply the current Course State travel effect
Failure: Apply one Course State worse for this turn only
Critical Failure: Apply one Course State worse for this turn only and reduce Course Meter by 1
If the ship is already Lost, one Course State worse still uses Lost
Sailor Background, Pirate Background, or Water Vehicles proficiency rolls with advantage
Navigator's Tools and Cartographer's Tools do not grant advantage on Man the Helm
Broken Mast: no Helm check; action still costs Labor, but Helm applies no travel tick change
Broken Rudder with functional Mast: no Helm check; website randomly applies True Course, Drifting, Off Course, or Lost travel ticks, ignoring Course Meter
Broken Mast and Broken Rudder: use Broken Mast behavior; no random rudder movement

Labor and Exhaustion Rules
Each character can accumulate up to 3 Labor Points safely. Taking a Labor action while already at 3 Labor requires a Constitution Save with an increasing DC based on this table
Labor Amount	CON DC
4		12
5		14
6		16
7		18
8		20
Failure: Gain 1 Level of Exhaustion
Success: No Exhaustion but still gain the Labor Point

Water System
At the end of every turn: Water Level += Water ingress + # of Active Leaks
Minimum Water Ingress starts at 2 and cannot be lowered
Actions can be taken to lower the water level
